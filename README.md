# TI Reminder Calendar exercise

## Indications

To run/start the application, the command must be executed:

```sh
npm run dev
```

To build the application, the command must be executed:

```sh
npm run build
```

To run the tests, the command must be executed:

```sh
npm run test
```

## Tech Stack

- Node v20
- Vite
- Vitest
- React
- TypeScript
